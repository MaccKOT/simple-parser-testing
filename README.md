
# 📦 Парсер товаров Горздрав (Gorzdrav)

Этот проект представляет собой скрипт для сбора информации о товарах с сайта интернет-аптеки [gorzdrav.org](https://gorzdrav.org). Скрипт написан на Node.js с использованием библиотеки Playwright для автоматизации браузера.

## 🚀 Возможности

- **Автоматический сбор данных:** Извлекает название, цены (текущую и без скидки), производителя, страну, ссылку на изображение, статус рецептурного препарата и наличие.
- **Умная пагинация:** Автоматически определяет количество страниц в категории и проходит по всем ним.
- **Гибкие режимы работы:**
  - **Headless (фоновый):** Работает без графического интерфейса, быстро и незаметно.
  - **Headed (визуальный):** Открывает окно браузера, позволяя наблюдать за процессом (полезно для отладки).
- **Использование системного браузера:** По умолчанию пытается использовать установленный Microsoft Edge, что снижает риск блокировок со стороны сайта.
- **Экспорт в CSV:** Сохраняет результаты в файл формата CSV с кодировкой UTF-8 (с BOM), корректно отображаемый в Excel.

## 📋 Требования

- **Node.js** (версия 14 или выше)
- **npm** (идет в комплекте с Node.js)
- **Microsoft Edge** (опционально, используется по умолчанию для максимальной схожести с обычным пользователем)

## ⚙️ Установка

1. **Создайте папку проекта и перейдите в нее:**

   ```bash
   mkdir gorzdrav-parser
   cd gorzdrav-parser
   ```


2. **Инициализируйте проект и установите зависимости:**

   ```bash
   npm init -y
   npm install playwright
   ```

3. **Скачайте браузер для Playwright (если не установлен Edge или хотите использовать Chromium):**

   ```bash
   npx playwright install chromium
   ```

4. **Поместите файл скрипта** (например, `parser.js`) в папку проекта.

## 🏃 Запуск

Скрипт поддерживает управление режимом работы через аргументы командной строки.

### 1. Фоновый режим (по умолчанию)

Браузер работает скрытно. Это быстрый и эффективный режим для регулярного использования.

```bash
node parser.js
```

### 2. Режим с интерфейсом (для отладки)

Открывается реальное окно браузера. Используйте этот режим, если нужно проверить, как скрипт взаимодействует с сайтом, или если сайт блокирует headless-запросы.

```bash
node parser.js --headed
```

## 📄 Выходные данные

Результат работы скрипта сохраняется в текущей папке в файл вида `gorzdrav_html_ГГГГ-ММ-ДД.csv`.

### Структура CSV файла:

| Колонка           | Описание                                  |
| :---------------- | :---------------------------------------- |
| **ID**            | Идентификатор товара (извлекается из URL) |
| **Название**      | Наименование товара                       |
| **Ссылка**        | Полный URL страницы товара                |
| **Изображение**   | Ссылка на изображение товара              |
| **Цена**          | Текущая цена (со скидкой, если есть)      |
| **ЦенаБезСкидки** | Изначальная цена (до применения скидки)   |
| **Рецептурный**   | Требуется ли рецепт (Да/Нет)              |
| **Производитель** | Название компании-производителя           |
| **Страна**        | Страна производства                       |
| **ВНаличии**      | Доступен ли товар для покупки (Да/Нет)    |

## ⚠️ Важные особенности реализации

- **Защита от ботов:** Скрипт устанавливает `User-Agent` имитирующий реальный браузер Edge и использует системный браузер (`channel: 'msedge'`), чтобы минимизировать шансы на блокировку.
- **Оптимизация скорости:** В фоновом режиме блокируется загрузка изображений, видео и шрифтов для экономии трафика и времени.
- **Парсинг HTML:** Данные извлекаются непосредственно из HTML-разметки отображенной страницы, а не из внутренних JSON-объектов, что делает парсер более устойчивым к изменениям во внутренней структуре данных сайта.
